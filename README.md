# pipeline.run — Sanket Kumar's Portfolio

A data engineering portfolio designed as a live pipeline orchestration dashboard.

**Built with:** HTML, CSS, JavaScript (zero dependencies, single file)
**Hosted on:** GitHub Pages

## 🔧 Updating Content

All content lives in `index.html`. Search for these placeholders to customize:

| Placeholder | What to replace with |
|---|---|
| `#YOUR-CERT-LINK` | Your Microsoft/Databricks certification URLs |
| `#YOUR-CREDLY-LINK` | Your Credly badge URLs |
| `https://github.com/SanketK/your-repo-name` | Your actual GitHub project repo URLs |
| `href="#"` on experience cards | Case study or blog post links |
